# win_batch
Windows batch collection

| 파일명             | 설명                         |
| ------------------ | ---------------------------- |
| devenv.bat         | 개발환경 환경변수 서정       |
| reskit.bat         | windows 설치 설정 모음       |
| bootsel.bat            | Boot manager 선택기          |
| autorun.cmd        | 명령 프롬프트 자동 실행 배치 |
| vscode_gitlens.bat | VSCode GitLens 라이선스 등록 |
